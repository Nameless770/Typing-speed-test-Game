An interactive Typing Speed Test Game built using HTML, CSS, and Vanilla JavaScript.
The game challenges players to type randomly generated words within a limited time based on the selected difficulty level.

This project focuses on improving typing speed while practicing JavaScript concepts such as DOM manipulation, timers, game logic, and dynamic content updates.

The player clicks Start Playing.

The game continues until all words are completed or time runs out.

A final result message is displayed at the end.
